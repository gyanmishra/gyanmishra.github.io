How to run jupyter notebook from remote server

Step1 : Start jupyter notebook in remote server

nohup jupyter notebook --no-browser --port=8889

Step2 : Connect from remote server

ssh -N -f -L localhost:8889:localhost:8890 remote_username@remote_ip

Show running jupyter notebook

jupyter notebook list

Select column 1 to 4 from data frame using lapply
