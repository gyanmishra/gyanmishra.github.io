Simple steps to create your own GitHub webpage

Apr 8, 2020 2 min read R

This guide will help you to create your own GitHub Page from scratch using R package blogdown.

Note : This tutorial assumes that you some background knowlege of R.

So the first step is to install blogdown package.

# install
install.packages("blogdown")
#load the library
library(blogdown)

Before getting started check whether hugo is installed in your system or not. If not, install using below command.

sudo apt-get install hugo

After successful installation of package blogdown and hugo you are ready to build your webpage. you can choose your theme of interest from here.

# set your working directory where you want to build your site.
setwd("github_page")
# Here I have used hugo-academic theme version 4.2.5
blogdown::new_site(dir=".",format = "toml",theme = "gcushen/hugo-academic@v4.2.5")

The folder “github_page” will be populated with several files required for bulding webpage as you can see below.

> list.files()
[1] "config"      "config.toml" "content"     "index.Rmd"   "public"      "resources"   "static"      "themes"

To render the content of the “github_page” folder as webpage use following codes.

# buld webpage to another folder (e.g render)
blogdown::build_dir("render")
options(blogdown.publishDir="render")
blogdown::build_site()

Next, serve the hugo template that is built from above lines of code.

Here you go.

blogdown::serve_site()

The “render” directory will pe populated with the content of the webpage. Modify the markdown files and .toml inside the “render” folder as per your requirement and your webpage will be ready to publish.

After you have modified the conent and ready to go online, create a repoository at github with extension “.github.io” (for e.g your_webpage_name.github.io).

Now the final step is to push the content of the “render” folder to github repositiry that we created above.

# open terminal
cd render
# make sure that git is already installed in your system.
git init
git add .
git commit -m "First commit"
git remote add origin https://github.com/your_webpage_name/your_webpage_name.github.io
git push orginal master

Congrats you have succesfully created your GitHub webpage !
